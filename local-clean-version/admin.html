批量导入题目

支持Excel和JSON格式的题库导入

Excel格式要求

Excel文件需要包含以下列（列名可以是中文或英文）：

  • 题目 (或 question): 题目内容
  • 分类 (或 category): 题目分类
  • 类型 (或 type): single(单选) 或 multiple(多选)
  • 选项A (或 A, optionA): 第一个选项
  • 选项B (或 B, optionB): 第二个选项
  • 选项C (或 C, optionC): 第三个选项
  • 选项D (或 D, optionD): 第四个选项
  • 答案 (或 answer): 正确答案(如A、B、ABC等)
  • 解析 (或 explanation): 答案解析(可选)

JSON格式示例

[
  {
    "category": "疾病",
    "type": "single",
    "question": "题目内容",
    "optionA": "选项A",
    "optionB": "选项B", 
    "optionC": "选项C",
    "optionD": "选项D",
    "answer": "B",
    "explanation": "解析内容"
  }
]

AI API配置

配置通义千问API，用于生成智能考试分析报告

AI API配置说明

支持DeepSeek和通义千问两种API：

推荐：DeepSeek API - 国内直连，无需代理，稳定快速

DeepSeek配置示例：

{
  "provider": "deepseek",
  "url": "https://api.deepseek.com/v1/chat/completions",
  "model": "deepseek-chat",
  "key": "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
}
注意：DeepSeek API密钥格式为 sk- 开头的字符串

通义千问配置示例：

{
  "provider": "qwen",
  "url": "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions",
  "model": "qwen-turbo",
  "key": "您的通义千问API密钥"
}
  • provider: API提供商标识
  • url: API服务地址（使用OpenAI兼容格式）
  • model: 模型名称
  • key: API密钥

获取API密钥：

• DeepSeek: 访问 DeepSeek开放平台

• 通义千问: 访问 阿里云DashScope控制台
